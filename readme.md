# Backend

## Install ChromDB

```bash
pip install chromadb
```

## Install Backend Packages

```bash
npm i
```

## Start The ChromDB Server

```bash
chroma run --path /db_path
```

## Start The backend devlopment server

```bash
npm run dev
```

# Frontend

## Go to frontend folder

```bash
cd ./frontend
```

## Start Devlopment Server

```bash
npm start
```
